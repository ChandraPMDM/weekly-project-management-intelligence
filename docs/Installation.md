# Installation Guide

## Prerequisites

- n8n Cloud
- OpenAI API Key
- Gmail Account

## Installation

1. Clone the repository.

2. Import the workflow into n8n.

3. Configure OpenAI credentials.

4. Configure Gmail OAuth.

5. Update recipient email.

6. Publish workflow.

7. Execute manual test.

8. Activate Schedule Trigger.

---

## Credentials

This repository intentionally excludes credentials.

You must configure:

- OpenAI
- Gmail OAuth

inside your own n8n instance.
