# 📬 Weekly Project Management Intelligence

> An AI-powered weekly Project Management newsletter built with **n8n**, **OpenAI GPT-4.1**, **RSS feeds**, JavaScript, and Gmail.

![n8n](https://img.shields.io/badge/n8n-Automation-orange)
![OpenAI](https://img.shields.io/badge/OpenAI-GPT--4.1-green)
![License](https://img.shields.io/badge/License-MIT-blue)

## 🚀 Overview

Weekly Project Management Intelligence automatically gathers the latest Project Management news, filters relevant articles, generates an AI-powered newsletter, and emails it every weekend.

The solution is designed for:

- Technical Delivery Managers
- Project Managers
- Scrum Masters
- PMO Professionals
- Program Managers
- Agile Coaches

---

## ✨ Features

- Weekly automated execution
- RSS news aggregation
- AI-generated newsletter
- Gmail integration
- HTML email output
- OpenAI GPT integration
- JavaScript preprocessing
- Low-code automation using n8n
